What is Morse Code

Morse code is a system of dots (short signals) and dashes (long signals) used to represent letters, numbers, and symbols. This system allows for easy transmission of information using sound, light, or written symbols.

Basic Rules

  • Dots are short signals
  • Dashes are long signals
  • Use space to separate characters
